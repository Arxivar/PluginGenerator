***INSTRUCTIONS
The libs folder is a space dedicated to containing any external libraries (JS) useful for the operation of the plugin.

Use this folder to add .js or .css files related to external dependencies that you use within your plugin. You can reference these files directly in the HTML using the <script> tag

Example:

<link rel="stylesheet" href="Scripts/plugins/*MyPluginFolder*/libs/externalFile.css" />
or
<script type="text/javascript" src="Scripts/plugins/*MyPluginFolder*/libs/externalFile.js" />
